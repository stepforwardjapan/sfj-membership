# 03. 集客LP 要件

## LPの役割
- 目的: 訪問者を **挑戦サポーター登録**（年3万円）まで運ぶ
- KPI: LP → 登録開始CVR / LP → 決済完了CVR

## 訴求軸

### メイン訴求
**「失敗できる社会を、一緒につくる。」**
- SFJは「思想コミュニティ型メディア」（ツール提供ではなく、世界観への共感で集める）

### サブ訴求（ターゲット別）
| ターゲット | コピー方向 |
|---|---|
| 個人向け | 「"やってみたい"を、"やってよかった"に。」 |
| 企業向け | 「"失敗は減点"から、"挑戦は評価"へ。」 |
| 社会的意義 | 「挑戦すると損をする社会、終わらせよう。」 |

## 必要セクション
- [x] ファーストビュー（ビジョン＋CTA「サポーターになる」）

- [x] ソリューション（SFJの3本柱: Podcast / サポーター制度 / プロジェクト）

- [x] サポーター特典一覧（7つ）
- [x] 料金（年3万円、個人法人共通）
- [x] 実績（クラファン171名・202%達成、2025年の登壇者8名）
- [x] ゲスト＆参加者の声
- [x] FAQ
- [x] 理事紹介（5名）
- [x] 特定商取引法に基づく表記リンク
- [x] 最終CTA

## サポーター特典（LPに掲載する7つ）
1. 公開収録（失敗の履歴書Podcast）への毎回無料参加
2. 毎回3名まで無料招待可能
3. SFJプロジェクトの立ち上げ権
4. コミュニティ内イベント企画の権限
5. SFJリアルイベントの割引 or 無料招待
6. 失敗の履歴書Podcastに1回ゲスト出演可能（事前面談あり）
7. Web等に名前・法人名の掲載（任意）

## 流入元
| チャネル | 優先度 | 備考 |
|---|---|---|
| Instagram | 高 | 「失敗研究所 by SFJ」として発信。リール週3 / カルーセル週2 |
| Facebook / LinkedIn | 中 | 既存の主力。DM集客に依存（7割がDM経由） |
| Podcast経由 | 高 | 番組の概要欄からLP誘導 |
| Peatix | 中 | イベント参加者→サポーター転換 |
| LINEオープンチャット | 中 | コミュニティ中間導線として設計中 |
| note | 低 | アーカイブ記事からの長期流入 |

### 導線設計（現行の分析から）
```
Instagram → LINEコミュニティ → イベント参加 → サポーター入会
Podcast → 概要欄リンク → LP → サポーター入会
DM/紹介 → LP → サポーター入会
```
※「イベントは入口ではなくゴール」にする方針
