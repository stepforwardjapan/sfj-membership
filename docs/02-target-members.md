# 02. ターゲット会員像

## 想定する会員像

### 挑戦サポーター（有料会員）
- **年齢層**: 30〜50代が中心
- **職業**: 経営者、リーダー、新規事業担当者、自治体職員、独立準備中の人
- **特徴**: 「このままじゃいけない」と思いながら一歩が踏み出せないビジネスパーソン
- **デバイス**: スマホ中心（Podcast視聴）、PC併用（管理画面・申込）

### 若者（無料参加枠）
- **年齢**: 18〜29歳
- **条件**: 公開収録に毎回無料で参加可能（事前申込のみ、コミュニティ所属不要）
- **目的**: 先輩の失敗を疑似体験し、挑戦の防衛本能のハードルを下げる

## ペルソナ

### ペルソナA: 組織内で動きたいリーダー
- 属性: 40代男性、大手企業の中間管理職
- ニーズ: 組織内で挑戦したいが周囲が保守的で進めづらい
- SFJに期待: 同じ志を持つ社外の仲間、具体的な失敗事例から学ぶ勇気
- 利用シーン: 月1回のPodcast視聴 + 公開収録参加、プロジェクトへの関与

### ペルソナB: 独立を検討中の個人
- 属性: 30代女性、転職または独立を検討中
- ニーズ: 一歩踏み出す勇気、成功者だけでない等身大のキャリア事例
- SFJに期待: 「失敗しても大丈夫」と思える仲間・コミュニティ
- 利用シーン: Instagram/Podcastで認知 → 公開収録参加 → サポーター入会

### ペルソナC: 若者（学生・若手社会人）
- 属性: 20代、大学生または新卒数年目
- ニーズ: 「やってみたい」が漠然とある、失敗が怖い
- SFJに期待: 先輩の実体験、番組運営への参加を通じた実践経験
- 利用シーン: 公開収録に無料参加、番組制作に学生スタッフとして参加

## 会員になる動機
1. **失敗を語れる安全な場所**: 普段の仕事・SNSでは言えない話ができる
2. **ゲストとの直接対話**: 公開収録の場で著名ゲストに質問できる
3. **挑戦者の仲間**: 異業種のチャレンジャーとの接点
4. **自分のプロジェクトを始められる**: SFJプロジェクトの発起人になれる
5. **ゲスト出演の機会**: 年1回、Podcast「失敗の履歴書」に出演可能

## 会員区分（確定: 2026年〜）

### 現行制度: 挑戦サポーター制度（Challenge Supporter Program）
| 項目 | 内容 |
|---|---|
| 年額 | **3万円（税込）** |
| 個人/法人区別 | **なし**（法人契約で経費処理OK） |
| 位置づけ | 「ファン」ではなく「共に挑戦文化をつくる当事者」 |

### 過去の制度変遷
| 時期 | モデル | 区分 |
|---|---|---|
| 2025年1月（設立時） | 3区分制 | 法人30万円 / 個人12万円 / アソシエイト3万円 |
| 2025年7月頃 | 2区分制 | 協賛30万円 / 会員3万円 |
| 2025年11月〜 | **統一制** | **挑戦サポーター 3万円（個人法人共通）** |

### 無料参加の導線
| 対象 | 条件 |
|---|---|
| 若者（18〜29歳） | 誰でも毎回無料参加OK（事前申込のみ） |
| オンラインコミュニティ | 過去イベント参加者 / クラファン支援者 / 理事推薦（招待制） |
